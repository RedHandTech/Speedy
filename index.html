Speedy

A concise reactive Swift property observation framework

Overview

What Is Reactive Programming?

Reactive programming, or observational programming, provides the developer with the ability to be instantaneously informed of a change in value of a variable throughout an application's lifetime.


Why Use Reactive Programming?

Reactive programming, and Speedy in particular, can be very useful when adopting an MVVM architecture for your applications. Speedy enables you to easily update your view layer based on changes in your view model.

Getting Started

It's easy to get started with Speedy. You can use Carthage, Cocoapods, Swift Package Manager or simply donwnload and build from source.
